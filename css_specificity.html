CSS Specificity คืออะไร?

CSS Specificity คือระบบคำนวนค่าความสำคัญของ CSS style เพื่อให้ browser รู้ว่าต้องนำ style ใดมาแสดงผล โดย browser จะนำ attribute ของ style ที่มีค่าความสำคัญสูงสุดมาแสดงผล style แต่ละแบบมีลำดับความสำคัญต่างกัน เรียงลำดับค่าความสำคัญจากมากไปน้อยได้ดังนี้

1. !important
2. inline style
3. ID selectors
4. class selectors
5. type selectors

!important

มีความสำคัญสูงสุด นำไปใส่ไว้ที่ style ใด หมายความว่าต้องใช้ style นั้นเสมอ เช่น

//CSS
.blue {color: blue !important;}
//HTML
<p class="blue" style="color: red">text</p>

p จะแสดงผลเป็นสีฟ้า ถึงแม้ว่า inline style จะระบุให้ใช้สีแดงก็ตาม

inline style

คือการใส่ style ไว้ใน html tag มีความสำคัญรองลงมาจาก !important เช่น

//CSS
#blue {color: blue;}
//HTML
<p id="blue" style="color: red">text</p>

p จะแสดงผลเป็นแดงตามที่ระบุใน inline style

ID selectors

มีความสำคัญรองจาก inline style ใช้สำหรับจัดกลุ่ม style attribute ให้กับ element

//CSS
#blue {color: blue;}
.red {color: red;}
//HTML
<p id="blue" class="red">text</p>

p จะแสดงผลเป็นสีฟ้า เนื่องจาก id มีค่าความสำคัญสูงกว่า class

class selectors

มีความสำคัญรองจาก ID ใช้สำหรับจัดกลุ่ม style attribute ให้กับ element เช่นเดียวกับ ID เช่น

//CSS
.blue {color: blue;}
p {color: red;}
//HTML
<p class="blue">text</p>
<p>text2</p>

text จะแสดงผลเป็นสีฟ้า ในขณะที่ text2 จะแสดงผลเป็นสีแดง

type selectors

ใช้สำหรับกำหนด style attribute ให้กับ tag ต่างๆ เช่น

//CSS
p {color: blue;}
//HTML
<p>text</p>

text จะแสดงผลเป็นสีฟ้า

How to calculate CSS Specificity

ถ้ามีค่าความสำคัญเท่ากันเลือก CSS selector ที่อยู่ด้านล่างมาแสดงผล
.classname h2 a { } คำนวณค่าความสำคัญโดยนำค่าความสำคัญของ classname + h2 + a แล้วนำมาเปรียบเทียบกันตามปกติ
selector ที่มีค่าความสำคัญมากกว่าจะถูกนำมาแสดงผล
